The model built and used for detecting and predicting brain tumor is a 2D CNN with one hidden layer. It is built with Tensorflow and Keras. The input images (data set is described in the "Data Description" tab) are 240x240x1 the 1 indicates 1 channel and converted into grayscale. The output is just a 1D array the length of the samples (ex. if input 100 images, it would output 100 predictions of 1 or 0 for whether or not there is a tumor).
